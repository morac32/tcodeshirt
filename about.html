About This Project

Electronic commerce is one of the most used ways to purchases goods and services. It is done over the internet and is done either business-to-business, or business-to-consumer. We want to create a website that sells T-Shirts to consumers. The website will allow consumers to browse through all the T-Shirts that are for sale. Consumers will be able to search for T-Shirts by size. Issues associated with this kind of website are the variety in the T-Shirts, so it's important to be able to filter the T-Shirts by the different sizes. Hopefully, this website will allow consumers to easily navigate to the type of T-Shirts that they want to purchase.
